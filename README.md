# 📚 ComicReader — SillyTavern 漫画阅读器扩展

一个为 [SillyTavern]打造的精致漫画阅读器插件。支持多格式导入、滚动/翻页双模式、10 种主题配色、缩略图目录、自定义外观等功能。


## ✨ 功能特点

- **多格式导入** — 支持批量图片（JPG/PNG/WebP/GIF）、ZIP 压缩包、PDF 文件
- **双阅读模式** — 滚动模式（连续浏览）与翻页模式（单页切换）自由切换
- **图片适配** — 适应宽度 / 适应高度 / 原始大小，双击图片可快速切换
- **翻页方向** — 支持从左到右（普通漫画）和从右到左（日本漫画）
- **缩略图目录** — 网格化页面缩略图，快速定位跳转
- **10 种主题** — 樱花粉、嫩黄、淡绿、毛玻璃白/黑、夜樱、深海蓝、落日橘、薰衣草、摩卡棕
- **可拖拽悬浮窗** — 自由移动、缩放面板，带猫耳悬浮球收纳
- **自定义外观** — 悬浮球头像、边框纹理、阅读背景图、封面图片
- **本地存储** — 基于 IndexedDB，数据完全离线，隐私安全
- **备份还原** — 一键导出/导入 JSON 备份文件，支持跨设备迁移
- **移动端适配** — 触摸滑动翻页、长按编辑封面、响应式布局

## 📦 安装方式

### 方法一：通过 SillyTavern 内置安装（推荐）

1. 打开 SillyTavern，进入 **Extensions** 面板
2. 点击 **Install Extension**
3. 输入本仓库地址：https://github.com/yangtfei204-hub/ComicReader.git
4. 点击安装，等待完成后刷新页面
